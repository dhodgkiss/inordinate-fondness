January 1, 2018 the Anxious Analyst

Introduction When preparing data frames for input into analyses, a common first step is to restructure the layout of the data so that you have all the necessary variables as columns and all the relevant values for each observation in rows. However, recorded data is often in a format that suits data collection, but which is unsuitable for data analysis. At times like these, rearranging the data, or ‘reshaping’ the data frame, is a necessary prerequisite for carrying out the analyses.
